# MHR 配装器

面向《怪物猎人崛起：曙光》和《怪物猎人荒野》玩家的中文静态配装网站。功能包含：

- 技能目标、武器模板、护石技能和孔位输入
- 14 种武器类型模板与武器孔位输入
- 首页双版本入口：怪物猎人崛起：曙光配装器、怪物猎人荒野配装器
- 完整装备、武器、技能和装饰品 JSON 数据
- 从全装备库筛选候选后进行装备组合与装饰品补技能
- 荒野武器珠、防具珠孔位类型区分
- 指定头身腕腰脚装备、排除防具、排除装饰品
- 逐孔手动装珠，荒野武器孔与防具孔按装饰品类型过滤
- 后台 worker 搜索、进度显示和停止搜索；搜索范围可在智能推荐、深度搜索、全量搜索之间切换
- 推荐配装结果、剩余孔位、缺失技能提示、锁定/排除、应用到装备栏、排序、收藏和复制方案
- 二维码分享当前配装条件、收藏方案 JSON 导入导出、单套方案 PNG 图片导出
- SEO 标题、描述、Open Graph、JSON-LD、robots、sitemap
- Google AdSense 脚本与广告位占位
- 原创首屏视觉资产，未使用官方素材

## 本地预览

```powershell
python -m http.server 8000
```

打开 `http://localhost:8000/`。

荒野配装器：`http://localhost:8000/wilds.html`。

## 上线前要改

1. 把 `index.html`、`privacy.html`、`robots.txt`、`sitemap.xml` 中的 `https://mhr-build.example.com/` 替换为真实域名。
2. 把 `ca-pub-XXXXXXXXXXXXXXXX` 替换为你的 Google AdSense 发布商 ID。
3. 把 `data-ad-slot` 替换为 AdSense 后台创建的广告单元 ID。
4. AdSense 申请前建议补充“关于本站/联系我们”页面，并持续增加原创配装内容或装备数据库页面。

默认搜索会从完整防具库中按目标技能、孔位和防御筛出每个部位最相关的候选，再进行组合计算，并按目标完成度、剩余孔位和防御排序。需要更深结果时可切换为深度搜索；全量搜索会把全部未排除装备交给 worker 逐组合计算，并保留进度和停止按钮。
